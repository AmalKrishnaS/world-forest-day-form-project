# world-forest-day-form-project

`Overview` 
A HTML form for signing up for a event for planting trees on the world forest day. This project is done to learn about html forms and how to style them.

`Things I learned` 
- HTML forms
- Styling forms
- How to achieve desired sizes for box element without giving height property which causes 
  undesired outputs when using with flexbox.
- Understood flexboxes better
- Learned about psuedo classes.
- Learned how to use box-shadow and outline
- Understood css much better
